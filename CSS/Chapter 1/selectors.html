Selectors: A CSS selector selects the HTML element(s) you want to style.

1. Element Selector: The element selector selects HTML elements based on the element name. Example - To select all paragraphs in a webpage use "p { }"

Element Selector

2. Id Selector : Uses the id attribute of an HTML element to select a specific element. Example - To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Id Selector

3. Class Selector : Selects HTML elements with a specific class attribute. Example - To select elements with a specific class, write a period (.) character, followed by the class name.

Class Selector

4. Universal Selector: The universal selector (*) selects all HTML elements on the page. 5. Attribute Selectors: The [attribute] selector is used to select elements with a specified attribute. Example - p elements with a target attribute.

Target Selector
